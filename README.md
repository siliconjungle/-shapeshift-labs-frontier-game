# Frontier Game

Reserved placeholder for `@shapeshift-labs/frontier-game`.

This repository is intentionally a placeholder. It reserves the package and source
repository name for a future game-facing API over Frontier realtime and state
packages. It does not contain production APIs, implementation code, benchmark
claims, or release-ready package contents yet.

Planned scope:

- entity, component, player, and room vocabulary
- spawn, despawn, ownership, and replication policy helpers
- ergonomic game-state APIs over Frontier realtime primitives
- adapters for application engines without owning rendering or physics

This package is intended to stay above `@shapeshift-labs/frontier-realtime`.
Renderer, physics, editor UI, and app-specific gameplay code should remain
outside lower Frontier packages.
